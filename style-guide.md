# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Tablet: 768px
- Desktop: 1024px


## Colors

## Typography

### Font Family

- Family: [Space Mono](https://fonts.google.com/specimen/Space+Mono)
- Weights: 700 (Bold)

### Text Presets

- **Text Preset 1**: Space Mono Bold, 48px, 71px Line Height, -1px Letter Spacing
- **Text Preset 2**: Space Mono Bold, 32px, 47px Line Height, -0.67px Letter Spacing
- **Text Preset 3**: Space Mono Bold, 24px, 36px Line Height, 0px Letter Spacing
- **Text Preset 4**: Space Mono Bold, 20px, 30px Line Height, 0px Letter Spacing
- **Text Preset 5**: Space Mono Bold, 16px, 24px Line Height, 0px Letter Spacing
- **Text Preset 6**: Space Mono Bold, 13px, 19px Line Height, 0px Letter Spacing

## Components

### Tip Percentage Buttons

- **Default State**: Background `Green 900`, Text `White`
- **Active State**: Background `Green 400`, Text `Green 900`
- **Hover State**: Background `Green 200`, Text `Green 900`

### Input Fields

- **Default State**: Background `Grey 50`, Border `None`
- **Active/Focus State**: Background `Grey 50`, Border `Green 400`
- **Error State**: Background `Grey 50`, Border `Orange 400`, Error Text `Orange 400` ("Can't be zero")

### Result Card

- **Card Background**: `Green 900`
- **Result Numbers (Tip & Total)**: `Green 400`
- **RESET Button (Disabled)**: Background `Green 800` (or reduced opacity), Text `Green 900`
- **RESET Button (Active)**: Background `Green 400`, Text `Green 900`
- **RESET Button (Hover)**: Background `Green 200`, Text `Green 900`
